Find the parametric form for the plane $\mathbb{R}^3$ given by the equation 
\[
6x-6y-z=7
\]

\[
x=1, y=1
\]

\[
6x-6y-z=7
\]
we have z = -7
So \[
P = (1,1,-7)
\]

\[

\[
6x-6y-z=7

we have z = -1
So
\[
R = (1,0,-1)

\[
